\noindent
Running the code without any arguments does the serial integration for a default value of \textit{-N=10000}. With the attribute \textit{-p} or \textit{--parallel}
the integration is run on parallel. For a comparison of exponentially growing $N$ use \textit{-c} or \textit{-comp}.
To visualize these results use \textit{-v} or \textit{-visualize}. All can be found in the help and is at the moment executed sequentially in the jobscript.
